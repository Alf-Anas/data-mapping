### ABOUT

This Repository Data Mapping is a dataset to map other data according to this data mapping. 
The world country mapping data contain ISO2, ISO3, CONTINET, Country Name, and other data that can be used to map country data with other data.

The Indonesia Provinsi and Kabupaten/Kota (KabKot) is for mapping province and kabkot data in Indonesia using KODE_PROV and KODE_KK according Kemendagri data.
